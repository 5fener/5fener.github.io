  • #include<math.h>

    1. abs函数
      函数定义：int abs（int i）；
      函数功能：求整数的绝对值

    2. labs函数
      函数定义：long labs(long n);
      函数功能：求长整数的绝对值

    3. fabs函数
      函数定义：double fabs(double x);
      函数功能：返回浮点数的绝对值

    4. sin函数
      函数定义：double sin(double x);
      函数功能：求解正弦

    5. cos函数
      函数定义：double cos(double x);
      函数功能：求解余弦

    6. tan函数
      函数定义: double tan(double x);
      函数功能：求解正切

    以下举例给出1~3的程序代码：

…

#include<stdio.h>
#include<math.h>

int main()
{
    int iAbsoluteNumber;
    int iNumber1 = -12;
    iAbsoluteNumber = abs(iNumber1);
    printf("%d\n", iAbsoluteNumber);

    long iResult;
    long iNumber2 = -1234567890L;
    iResult = labs(iNumber2);
    printf("%ld\n", iResult);

    double fFloatResult;
    double fNumber = -1234.0;
    fFloatResult = fabs(fNumber);
    printf("%lf", fNumber);

    return 0;
}

    1. isalpha函数
      函数定义：int isalpha(int ch);
      函数功能：检测字符是否为字母

    2. isdigit函数
      函数定义：int isdigit(int ch);
      函数功能：检测字符是否为数字

    3. isalnum函数
      函数定义：int isalnum(int ch);
      函数功能：检测字符是否为字母或数字

    以下举例给出程序代码：

‘’’

#include<stdio.h>
#include<ctype.h>

void SwitchShow(char c);

int main()
{
    char cCharPut;
    char cCharTemp;

    printf("First enter:");
    scanf_s("%c", &cCharPut,sizeof(cCharPut));
    SwitchShow(cCharPut);
    cCharTemp = getchar();

    printf("Second enter:");
    scanf_s("%c", &cCharPut,sizeof(cCharPut));
    SwitchShow(cCharPut);
    cCharTemp = getchar();

    printf("Third enter:");
    scanf_s("%c", &cCharPut,sizeof(cCharPut));
    SwitchShow(cCharPut);

    return 0;
}

void SwitchShow(char cChar)
{
    if (isalpha(cChar))
        printf("You entered a letter of  the alphabet %c\n", cChar);
    if (isdigit(cChar))
        printf("You entered the digit %c\n", cChar);
    if (isalnum(cChar))
        printf("You entered the alphanumuric character %c\n", cChar);
    else
        printf("You entered the character is not alphabet or digit:%c\n", cChar);
}

输入输出略解

发表于 2021-04-07 更新于 2021-04-10
本文字数： 926 阅读时长 ≈ 1 分钟
  • ###输入###
* `scanf`
* 函数原型:`int scanf(const char *format, ...)` *
* 函数从标准输入流 stdin 读取输入，并根据提供的 format 来浏览输入 *
* (format 可以是一个简单的常量字符串，但是您可以分别指定 %s、%d、%c、%f 等来输出或读取字符串、整数、字符或浮点数) *
* scanf遇到空格、回车和Tab键都会认为输入结束，所有它不能接收空格 *

* `gets`
* 函数原型：`char *gets(char *s)` *
* 从标准输入接收一串字符，遇到'\n'时结束，但不接收'\n'，把 '\n'留存输入缓冲区；把接收的一串字符存储在形式参数指针指向的空间，并在最后自动添加一个'\0' *
* gets可以接收空格 *

* `getchar`
* 函数原型: `int getchar(void)` *
* 从标准输入接收一个字符返回，在同一个时间内只会读取一个单一的字符，多余的字符全部留在输入缓冲区，什么时候结束由程序员自己约定结束符，一般选择\n为结束符，当然也可以是任意其他字符 *

* `fgets`
* 函数原型：`char *fgets(char *s, int n, FILE *stream)` *
* 从文件指针stream中读取字符，存到以s为起始地址的空间里，直到读完N-1个字符，或者读完一行 *
* 我们平时可以这么使用：fgets(str, sizeof(str), stdin);其中str为数组首地址，sizeof(str)为数组大小，stdin表示我们从键盘输入数据 *
  • ###输出###
* `printf`
* 函数原型:`int printf(const char *format, ...)` *
* 函数把输出写入到标准输出流 stdout ，并根据提供的格式产生输出 *

* `puts`
* 函数原型:`int puts(const char *s)` *
* 函数把字符串 s 和一个尾随的换行符写入到 stdout *

* `putchar`
* 函数原型:`int putchar(int c)` *
* 把字符输出到屏幕上，并返回相同的字符。这个函数在同一个时间内只会输出一个单一的字符。可以在循环内使用这个方法，以便在屏幕上输出多个字符 *

安装Hexo——Blog

发表于 2021-04-02 更新于 2021-04-15
本文字数： 479 阅读时长 ≈ 1 分钟

  1. 创立仓库

    先在GitHub上创立一个仓库

  2. 使用git安装

    建立一个Blog文件夹
    在Blog文件右键Git Bush Here并输入npm install -g hexo-cli命令安装Hexo
    输入hexo init命令初始化博客
    输入hexo g命令静态部署
    输入hexo s命令本地查看（在浏览器中输入https://local:4000)

  3. 将Hexo部署到Github

    回到Blog文件夹用记事本打开_config.yml文件
    下滑到文件底部，修改并填写如下内容：

    deploy: type: git repository: https://github.com//5fener//5fener.github.io.git /#你的仓库名

    在Blog文件打开Git Bush 安装git部署插件，即输入命令：
    npm install hexo hexo-deployer-git --save     注意空格
    输入hexo clean清除缓存文件和动态文件
    输入hexo g生成网站静态文件到默认设置的public文件夹中
    输入hexo d自动生成并部署到设定仓库
    可能延迟需要等待网站才会变化，可输入hexo s查看本地Blog是否改变
