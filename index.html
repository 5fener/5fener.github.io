W3school-HTML

发表于 2021-06-24
本文字数： 2.6k 阅读时长 ≈ 2 分钟

  1. HTML标签

    • html链接：<a href="http://www.w3school.com.cn">This is a link</a>
    • html图像：<img src="w3school.jpg" wedth="250" height="250" />
    • html标签建议使用小写


  2. HTML元素

    • 大多数html元素可以嵌套
    • html文档由嵌套的html元素构成
    • <p>元素定义html文档的段落
    • <body>元素定义html文档的主体
    • <br />空元素 换行


  3. HTML属性

    • 居中排列标题
      <h1>定义标题的开始。
      <h1 align="center"> 拥有关于对齐方式的附加信息
      ·背景颜色
      <body> 定义 HTML 文档的主体。
      <body bgcolor="yellow"> 拥有关于背景颜色的附加信息
    • 更多详细见完整的html参考手册
    • 建议使用小写属性
    • 当属性值本身含有双引号时，使用单引号包括属性值


  4. HTML标题

    • <h1>-<h6>定义，<h1>定义最大的标题，<h6>定义最小的标题，<h1>应作为主标题（最重要的），其次时<h2>(次重要),再其次<h3>，以此类推

    PS：浏览器会自动地在标题的前后添加空行
    PS：默认情况下，HTML 会自动地在块级元素前后添加一个额外的空行，比如段落、标题元素前后

    • <hr /> 标签在 HTML 页面中创建水平线，因此可用于（分割线）分隔内容（文章小节）
    • 通过使用例如<!--This is a comment-->用来给html代码插入注释，合理使用注释，提高代码可读性


  5. HTML段落

    • 浏览器会自动地在段落的前后添加空行。（<p> 是块级元素）
      使用空的段落标记 <p></p> 去插入一个空行是个坏习惯。用 <br /> 标签代替它
      不要忘记结束标签，即使大多数浏览器仍能正确显示
    • 如果希望在不产生一个新段落的情况下进行换行（新行），使用 <br /> 标签
      例如：<p>This is<br />a para<br />graph with line breaks</p>

    PS：<br /> 元素是一个空的 HTML 元素。由于关闭标签没有任何意义，因此它没有结束标签,对于<br>和<br />建议使用<br />(长远保障)

    • 当显示页面时，浏览器会移除源代码中多余的空格和空行。所有连续的空格或空行都会被算作一个空格
      需要注意的是，HTML 代码中的所有连续的空行（换行）也被显示为一个空格
      注意到当插入<br />换行前后空格或空行无作用


  6. HTML样式

    • style 属性的作用：提供了一种改变所有 HTML 元素的样式的通用方法
    • 在HTML 4中有若干标签和属性是被废弃的,请避免使用！！！
      应该避免下面这些标签和属性，使用样式代替：
    标签 描述
    <center> 定义居中的内容
    <font>和<basefont> 定义HTML字体
    <s>和<strike> 定义删除线文本
    <u> 定义下划线文本
    属性 描述
    <align> 定义文本的对齐方式
    <bgcolor> 定义背景颜色
    <color> 定义文本颜色
    • 样式实例-背景颜色
      <body style="background-color:yellow">
      <h2 style="background-color:red">This is a heading</h2>
    • 样式实例-字体、颜色、尺寸
      <p style="font-family:arial;color:red;font-size:20px;">A paragraph.</p>
    • 样式实例-文本对齐
      <h1 style="text-align:center">Thie is a heading</h1>


  7. HTML格式化

    • <pre></pre>预格式文本，保留了空格和换行，适合显示计算机代码
    • <abbr title="etcetera">etc.</abbr>
      <acronym title="World Wide Web">WWW</acronym>
      abbr和acronym都英文含义都有缩写的意思
      在某些浏览器中，当把鼠标移至缩略词语上时，title 可用于展示表达的完整版本
    • 如果浏览器支持 bi-directional override (bdo)，下一行会从右向左输出 (rtl)
      <bdo dir="rtl">Here is some Hebrew text</bdo>，此举用于改变文字输出方向
    • 标记删除文本（二十）和插入文本（十二）
      <p>一打有 <del>二十</del> <ins>十二</ins> 件。</p>
    • 文本格式化标签
    标签 描述
    <b> 定义粗体文本
    <big> 定义大号字
    <em> 定义着重文字
    <i> 定义斜体字
    <small> 定义小号字
    <strong> 定义加重语气
    <sub> 定义下标字
    <sup> 定义上标字
    <ins> 定义插入字
    <del> 定义删除字
    • 引用、引用和术语定义
    标签 描述
    <addr> 定义缩写
    <acronym> 定义首字母缩写
    <address> 定义地址
    <bdo> 定义文字方向
    <blockquote> 定义长的应用
    <q> 定义短的应用
    <cite> 定义引用、引证
    <dfn> 定义一个定义项目

  • #include<math.h>

    1. abs函数
      函数定义：int abs（int i）；
      函数功能：求整数的绝对值

    2. labs函数
      函数定义：long labs(long n);
      函数功能：求长整数的绝对值

    3. fabs函数
      函数定义：double fabs(double x);
      函数功能：返回浮点数的绝对值

    4. sin函数
      函数定义：double sin(double x);
      函数功能：求解正弦

    5. cos函数
      函数定义：double cos(double x);
      函数功能：求解余弦

    6. tan函数
      函数定义: double tan(double x);
      函数功能：求解正切

#include<stdio.h>
#include<ctype.h>

void SwitchShow(char c);

int main()
{
    char cCharPut;
    char cCharTemp;

    printf("First enter:");
    scanf_s("%c", &cCharPut,sizeof(cCharPut));
    SwitchShow(cCharPut);
    cCharTemp = getchar();

    printf("Second enter:");
    scanf_s("%c", &cCharPut,sizeof(cCharPut));
    SwitchShow(cCharPut);
    cCharTemp = getchar();

    printf("Third enter:");
    scanf_s("%c", &cCharPut,sizeof(cCharPut));
    SwitchShow(cCharPut);

    return 0;
}

void SwitchShow(char cChar)
{
    if (isalpha(cChar))
        printf("You entered a letter of  the alphabet %c\n", cChar);
    if (isdigit(cChar))
        printf("You entered the digit %c\n", cChar);
    if (isalnum(cChar))
        printf("You entered the alphanumuric character %c\n", cChar);
    else
        printf("You entered the character is not alphabet or digit:%c\n", cChar);
}

输入输出略解

发表于 2021-04-07 更新于 2021-04-10
本文字数： 926 阅读时长 ≈ 1 分钟
  • ###输入###
* `scanf`
* 函数原型:`int scanf(const char *format, ...)` *
* 函数从标准输入流 stdin 读取输入，并根据提供的 format 来浏览输入 *
* (format 可以是一个简单的常量字符串，但是您可以分别指定 %s、%d、%c、%f 等来输出或读取字符串、整数、字符或浮点数) *
* scanf遇到空格、回车和Tab键都会认为输入结束，所有它不能接收空格 *

* `gets`
* 函数原型：`char *gets(char *s)` *
* 从标准输入接收一串字符，遇到'\n'时结束，但不接收'\n'，把 '\n'留存输入缓冲区；把接收的一串字符存储在形式参数指针指向的空间，并在最后自动添加一个'\0' *
* gets可以接收空格 *

* `getchar`
* 函数原型: `int getchar(void)` *
* 从标准输入接收一个字符返回，在同一个时间内只会读取一个单一的字符，多余的字符全部留在输入缓冲区，什么时候结束由程序员自己约定结束符，一般选择\n为结束符，当然也可以是任意其他字符 *

* `fgets`
* 函数原型：`char *fgets(char *s, int n, FILE *stream)` *
* 从文件指针stream中读取字符，存到以s为起始地址的空间里，直到读完N-1个字符，或者读完一行 *
* 我们平时可以这么使用：fgets(str, sizeof(str), stdin);其中str为数组首地址，sizeof(str)为数组大小，stdin表示我们从键盘输入数据 *
  • ###输出###
* `printf`
* 函数原型:`int printf(const char *format, ...)` *
* 函数把输出写入到标准输出流 stdout ，并根据提供的格式产生输出 *

* `puts`
* 函数原型:`int puts(const char *s)` *
* 函数把字符串 s 和一个尾随的换行符写入到 stdout *

* `putchar`
* 函数原型:`int putchar(int c)` *
* 把字符输出到屏幕上，并返回相同的字符。这个函数在同一个时间内只会输出一个单一的字符。可以在循环内使用这个方法，以便在屏幕上输出多个字符 *

安装Hexo——Blog

发表于 2021-04-02 更新于 2021-04-15
本文字数： 479 阅读时长 ≈ 1 分钟

  1. 创立仓库

    先在GitHub上创立一个仓库

  2. 使用git安装

    建立一个Blog文件夹
    在Blog文件右键Git Bush Here并输入npm install -g hexo-cli命令安装Hexo
    输入hexo init命令初始化博客
    输入hexo g命令静态部署
    输入hexo s命令本地查看（在浏览器中输入https://local:4000)

  3. 将Hexo部署到Github

    回到Blog文件夹用记事本打开_config.yml文件
    下滑到文件底部，修改并填写如下内容：

    deploy: type: git repository: https://github.com//5fener//5fener.github.io.git /#你的仓库名

    在Blog文件打开Git Bush 安装git部署插件，即输入命令：
    npm install hexo hexo-deployer-git --save     注意空格
    输入hexo clean清除缓存文件和动态文件
    输入hexo g生成网站静态文件到默认设置的public文件夹中
    输入hexo d自动生成并部署到设定仓库
    可能延迟需要等待网站才会变化，可输入hexo s查看本地Blog是否改变
