W3school-HTML

发表于 2021-06-24 阅读次数：

1.HTML标签
·html链接：<a href="http://www.w3school.com.cn">This is a link</a>
·html图像：<img src="w3school.jpg" wedth="250" height="250" />
·html标签建议使用小写

2.HTML元素
·大多数html元素可以嵌套
·html文档由嵌套的html元素构成
·<p>元素定义html文档的段落
·<body>元素定义html文档的主体
·<br />空元素 换行

3.HTML属性
·居中排列标题
<h1>定义标题的开始。
<h1 align="center"> 拥有关于对齐方式的附加信息
·背景颜色
<body> 定义 HTML 文档的主体。
<body bgcolor="yellow"> 拥有关于背景颜色的附加信息
·更多详细见完整的html参考手册
·建议使用小写属性
·当属性值本身含有双引号时，使用单引号包括属性值

4.HTML标题
·<h1>-<h6>定义，<h1>定义最大的标题，<h6>定义最小的标题，<h1>应作为主标题（最重要的），其次时<h2>(次重要),再其次<h3>，以此类推
PS：浏览器会自动地在标题的前后添加空行
PS：默认情况下，HTML 会自动地在块级元素前后添加一个额外的空行，比如段落、标题元素前后
·<hr /> 标签在 HTML 页面中创建水平线，因此可用于（分割线）分隔内容（文章小节）
·通过使用例如<!--This is a comment-->用来给html代码插入注释，合理使用注释，提高代码可读性

5.HTML段落
·浏览器会自动地在段落的前后添加空行。（<p> 是块级元素）
使用空的段落标记 <p></p> 去插入一个空行是个坏习惯。用 <br /> 标签代替它
不要忘记结束标签，即使大多数浏览器仍能正确显示
·如果希望在不产生一个新段落的情况下进行换行（新行），使用 <br /> 标签
例如：<p>This is<br />a para<br />graph with line breaks</p>
PS：<br /> 元素是一个空的 HTML 元素。由于关闭标签没有任何意义，因此它没有结束标签,对于<br>和<br />建议使用<br />(长远保障)
·当显示页面时，浏览器会移除源代码中多余的空格和空行。所有连续的空格或空行都会被算作一个空格
需要注意的是，HTML 代码中的所有连续的空行（换行）也被显示为一个空格
注意到当插入<br />换行前后空格或空行无作用

6.HTML样式
·style 属性的作用：提供了一种改变所有 HTML 元素的样式的通用方法
·在HTML 4中有若干标签和属性是被废弃的，请避免使用！！！
应该避免下面这些标签和属性，使用样式代替：
标签 |描述
———————-|—————-
<center> |定义居中的内容
<font>和<basefont> |定义HTML字体
<s>和<strike> |定义删除线文本
<u> |定义下划线文本
属性 |描述
———————-|—————–
<align> |定义文本的对齐方式
<bgcolor> |定义背景颜色
<color> |定义文本颜色
·样式实例-背景颜色
<body style="background-color:yellow">
<h2 style="background-color:red">This is a heading</h2>
·样式实例-字体、颜色、尺寸
<p style="font-family:arial;color:red;font-size:20px;">A paragraph.</p>
·样式实例-文本对齐
<h1 style="text-align:center">Thie is a heading</h1>
7.HTML格式化
·<pre></pre>预格式文本，保留了空格和换行，适合显示计算机代码
·<abbr title="etcetera">etc.</abbr>
<acronym title="World Wide Web">WWW</acronym>
abbr和acronym都英文含义都有缩写的意思
在某些浏览器中，当把鼠标移至缩略词语上时，title 可用于展示表达的完整版本
·如果浏览器支持 bi-directional override (bdo)，下一行会从右向左输出 (rtl)；
<bdo dir="rtl">Here is some Hebrew text</bdo>，此举用于改变文字输出方向
·标记删除文本（二十）和插入文本（十二）
<p>一打有 <del>二十</del> <ins>十二</ins> 件。</p>
·文本格式化标签
